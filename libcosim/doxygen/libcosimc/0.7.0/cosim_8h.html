enum cosim_errc

Error codes.

Enumerator
COSIM_ERRC_UNSPECIFIED 

Unspecified error (but message may contain details).

COSIM_ERRC_ERRNO 

Error reported by C/C++ runtime; check errno to get the right code.

COSIM_ERRC_INVALID_ARGUMENT 

Invalid function argument.

COSIM_ERRC_ILLEGAL_STATE 

Function may not be called while in this state.

COSIM_ERRC_OUT_OF_RANGE 

Index out of range.

COSIM_ERRC_STEP_TOO_LONG 

The time step failed, but can be retried with a shorter step length (if supported by all slaves).

COSIM_ERRC_BAD_FILE 

An input file is corrupted or invalid.

COSIM_ERRC_UNSUPPORTED_FEATURE 

The requested feature (e.g. an FMI feature) is unsupported.

COSIM_ERRC_DL_LOAD_ERROR 

Error loading dynamic library (e.g. model code).

COSIM_ERRC_MODEL_ERROR 

The model reported an error.

COSIM_ERRC_SIMULATION_ERROR 

An error occured during simulation.

COSIM_ERRC_ZIP_ERROR 

ZIP file error.

◆ cosim_last_error_code()

cosim_errc cosim_last_error_code ( )

Returns the error code associated with the last reported error.

Most functions in this library will indicate that an error occurred by returning -1 or NULL, after which this function can be called to obtain more detailed information about the problem.

This function must be called from the thread in which the error occurred, and before any new calls to functions in this library (with the exception of cosim_last_error_message()).

Returns
The error code associated with the last reported error.

◆ cosim_last_error_message()

const char* cosim_last_error_message ( )

Returns a textual description of the last reported error.

Most functions in this library will indicate that an error occurred by returning -1 or NULL, after which this function can be called to obtain more detailed information about the problem.

This function must be called from the thread in which the error occurred, and before any new calls to functions in this library (with the exception of cosim_last_error_code()).

Returns
A textual description of the last reported error. The pointer is only guaranteed to remain valid until the next time a function in this library is called (with the exception of cosim_last_error_code()).
