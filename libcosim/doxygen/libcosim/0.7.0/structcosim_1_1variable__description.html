◆ reference

value_reference cosim::variable_description::reference

A numerical identifier for the value the variable refers to.

The variable reference must be unique within the model and data type. That is, a real variable and an integer variable may have the same value reference, and they will be considered different. If two variables of the same type have the same value reference, they will be considered as aliases of each other.
