Detailed Description

Interface for classes that represent function types and act as factory classes for function instances.

For more information about function types and instances, see the documentation for cosim::function.

virtual function_type_description cosim::function_type::description ( ) const
pure virtual

Returns a description of the function type.

Some aspects of a function description may depend on the values of certain parameters, which are only known after instantiation. Such fields are given a function_parameter_placeholder value in the returned object.

Implemented in cosim::vector_sum_function_type, and cosim::linear_transformation_function_type.
