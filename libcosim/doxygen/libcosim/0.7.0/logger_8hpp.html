Detailed Description

Defines the logger for this library, based on Boost.Log.

For convenience, this header also includes some Boost.Log headers that provide useful short-hand logging macros (like BOOST_LOG_SEV).

using cosim::log::severity_level = typedef boost::log::trivial::severity_level

Severity levels for logged messages.

This is currently an alias of boost::log::trivial::severity_level, mainly to take advantage of the built-in support that this type has in Boost.Log. However, this may change in the future, so it should be treated and used as if it were a separate enum whose enumerators are trace, debug, info, warning, error, and fatal.
