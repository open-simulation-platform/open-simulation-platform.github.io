Detailed Description

A URI reference.

A URI reference is an (absolute) URI if and only if it has a scheme component, i.e., the segment leading up to the first colon character. (For example: the http part of http://example.com).

cosim::uri::uri ( std::string  string)

Parses the contents of string.

string must either contain a valid URI reference or be empty. Passing an empty string is equivalent to calling the default constructor.

Complies with RFC 3986. The "authority" component is not validated or decomposed.

Exceptions
std::invalid_argumentif string does not contain a valid URI reference.

Composes a URI reference from its individual components.

Each component must conform to the rules described in RFC 3986. Beyond that, no validation is performed. (That is, no hostname lookup, no scheme-specific validation, and so on).

Passing an empty path and std::nullopt for all other components is equivalent to calling the default constructor.

Exceptions
std::invalid_argumentif any of the components have an invalid value.
