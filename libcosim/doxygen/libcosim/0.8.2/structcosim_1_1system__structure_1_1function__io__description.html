Detailed Description

A description of a function variable, including group and variable indices.

This is just a cosim::function_io_description bundled together with the indices of the variable and its group. (These indices are implicitly defined by the order of groups and variables in a cosim::function_description, and thus not an explicit part of each cosim::function_io_description.)
