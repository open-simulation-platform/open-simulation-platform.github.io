std::vector<function_parameter_description> cosim::function_type_description::parameters

The function's parameters.

Parameters are values which must be specified upon instantiation of a function with function_type::instantiate(). They typically determine properties of the function which cannot change during a simulation, such as the number and types of input/output variables.
