void cosim::real_time_timer::sleep ( time_point  currentTime)

Calls thread sleep for the amount of time it would take to keep real time.

If real time simulation is enabled, expected progress as well as elapsed time are calculated. Thread sleep is called for the amount of time it would take to synchronize against real time.

To be called at the tail end of each execution step.

Parameters
[in]currentTimeThe current simulation time.
