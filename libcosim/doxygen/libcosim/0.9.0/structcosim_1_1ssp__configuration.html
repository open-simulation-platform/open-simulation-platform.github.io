std::unordered_map<std::string, variable_value_map> cosim::ssp_configuration::parameter_sets

Named parameter sets.

This always contains at least one parameter set whose key is the empty string, which represents the default parameter values. (If the SSP configuration does not specify any parameter values, this set will be empty but nevertheless present.)
